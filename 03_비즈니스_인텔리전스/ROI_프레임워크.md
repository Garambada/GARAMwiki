# AI 도입 ROI 계산 프레임워크

> 마지막 업데이트: 2026-05-19

## 기본 공식

```
ROI = (절감 비용 + 창출 가치 - AI 도입 비용) / AI 도입 비용 × 100%

AI 도입 비용 = API 비용 + 개발 비용 + 운영 비용 + 교육 비용
```

## 비용 절감 계산법

| 항목 | 계산 방법 | 예시 |
|------|---------|------|
| 인력 시간 절감 | (절감 시간/월 × 시간당 인건비) | 20시간 × 50,000원 = 100만원/월 |
| 오류 감소 | (기존 오류율 - AI 도입 후 오류율) × 오류 처리 비용 | |
| 처리 속도 향상 | (기존 처리량 - 신규 처리량) × 단위 가치 | |

## API 비용 계산기

```
월 예상 비용 계산:

입력 토큰 수 = 평균 입력 길이(토큰) × 월 요청 수
출력 토큰 수 = 평균 출력 길이(토큰) × 월 요청 수

Claude Sonnet 기준:
월 비용 = (입력 토큰 × $3/1M) + (출력 토큰 × $15/1M)

예시: 월 10,000건, 평균 입력 500토큰, 출력 300토큰
= (5,000,000 × $3/1M) + (3,000,000 × $15/1M)
= $15 + $45 = $60/월 (약 8만원)
```

## 빠른 ROI 판단 기준

```
✅ 즉시 도입 추천 (ROI > 300%)
- 반복적인 단순 작업 (분류, 요약, 변환)
- 처리량이 많고 인건비가 높은 작업
- 오류 비용이 큰 작업

🟡 파일럿 후 결정 (ROI 100~300%)
- 복잡한 판단이 필요한 작업
- 기존 프로세스 변경이 필요한 작업

🔴 신중히 검토 (ROI < 100%)
- 창의성이 핵심인 작업
- 규제가 엄격한 영역
- 데이터 보안이 최우선인 영역
```
