# 벤더 선택 체크리스트

> AI 솔루션/서비스 벤더를 선택할 때 사용하는 평가 기준

## 1단계: 기본 자격 심사 (Pass/Fail)

```
□ 우리가 필요한 기능을 실제로 지원하는가?
□ 데이터 보안 및 개인정보 처리 정책이 수용 가능한가?
□ 국내 규제 (AI기본법, GDPR 등) 준수 여부
□ SLA (서비스 수준 협약) 조건이 요구사항 충족?
□ 레퍼런스 고객이 우리와 유사한 규모/업종인가?
```

## 4단계: 비즈니스 리스크

```
□ 벤더 락인 위험은 어느 정도인가?
□ 벤더가 갑자기 서비스 종료하면 대안이 있는가?
□ 가격 인상 시 전환 비용은?
□ 데이터가 모델 학습에 사용되는가?
```
