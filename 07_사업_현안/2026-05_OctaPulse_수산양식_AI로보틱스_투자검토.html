01 투자 개요 — 왜 지금 이 회사인가

OctaPulse는 양식업(hatchery 및 land-based farm)에서 반복적으로 수행되는 치어 선별, 기형 판별(deformity inspection), 품질 등급화 작업을 AI 비전과 로보틱스로 자동화하는 초기 B2B 기업이다. 기존 인프라에 카메라와 모듈형 하드웨어를 결합해 현장 도입 장벽을 낮추는 접근을 취하고 있으며, 초기 파일럿에서 90~95% 수준의 정확도를 제시하고 있다.

투자 관점에서 OctaPulse의 핵심은 단순한 검사 자동화가 아니라, 양식장 운영의 병목을 데이터 기반 의사결정으로 전환한다는 점이다. 검사의 표준화·속도 향상·품질 편차 감소가 동시에 발생하면 생산성 개선 효과가 직접적으로 매출과 원가 구조에 반영될 수 있다. 이것이 단순 검사장비 업체가 아닌 "양식장 운영 OS" 포지셔닝을 지향하는 이유다.

■ 투자 포지셔닝
초기 B2B AI/Robotics 투자 후보 — Conditional Positive. 문제 정의가 명확하고, 상용 고객 traction이 존재하며, 기술이 실제 운영 효율로 연결되는 구조를 갖췄다. 단, "기술 가능성"보다 "현장 재현성과 확장성"이 핵심 판단 기준이다.
02 문제 정의 — 양식업의 구조적 병목

양식업에서 가장 중요한 품질관리 단계 중 하나는 개체별 phenotyping과 deformity inspection인데, 이 작업은 여전히 숙련 인력이 육안으로 수행하는 경우가 많다. 공개 자료에 따르면 이 공정이 개체당 약 5분 걸리며, 생산량이 커질수록 인건비와 처리 병목이 급격히 증가한다.

이 문제는 단순한 자동화 니즈를 넘어, 어떤 개체를 주요 feeding stage로 진입시킬지 결정하는 생산 핵심 의사결정과 직결된다. 검사 속도와 정확도 개선은 feed waste 감소 → survival 개선 → yield 향상으로 이어지는 연쇄 임팩트를 갖는다. 중규모 농장 기준 한 코호트가 약 40만 마리에 달하고 그 중 절반이 등급 하락 요인을 가질 수 있다는 사실은 이 검사의 경제적 임팩트가 매우 크다는 것을 보여준다.

  • 01
    수작업 육안 검사 병목
    개체당 ~5분, 숙련 인력 의존 → 생산량 확대 시 비선형적 비용 급증
  • 02
    품질 선별 정확도 편차
    검사자·컨디션·종 차이에 따른 판별 기준 불일치 → 데이터 자산화 불가
  • 03
    Feeding stage 의사결정 지연
    부적합 개체 진입 → feed waste, 생존율 저하, yield 손실로 이어지는 연쇄 손실
  • 04
    데이터 부재로 breeding 최적화 불가
    개체 단위 표현형 데이터가 없어 broodstock 선발·유전 개선 루프 단절
03 제품과 기술 — 카메라에서 Delta Robot까지

OctaPulse의 제품은 상용 카메라 + 컴퓨터 비전 + 모듈형 하드웨어 + delta robotics 통합을 통해 fish inspection workflow를 자동화하는 구조다. 현재는 품질검사와 선별에 초점을 두고 있으며, 향후 breeding optimization과 health monitoring까지 확장 가능한 플랫폼을 지향한다.

📐 기술 스택
Computer Vision — broodstock phenotyping, juvenile deformity detection, weight estimation, grade classification

Data Flywheel — 각 deployment가 labeled image 증가 → multi-species dataset 확장 → cross-species generalization 강화

Delta Robotics — 자동 선별·분류 통합 (하드웨어 확장 로드맵)

Platform Vision — 검사 → feeding → health monitoring → breeding optimization 순차 확장
모델 정확도95%+
검사시간 단축률90%↓ (5min→30sec)
현장 배포 안정성 (검증 필요)미공개

기술적으로 단순한 모델 성능보다 데이터 축적 구조가 중요하다. 여러 종(trout, salmon, red drum, yellowtail 등)에서 확보한 데이터를 바탕으로 species generalization을 강화하고 있으며, 배포가 늘수록 모델 성능과 고객 락인 효과가 동시에 커지는 구조다.

04 경쟁 환경 — 좁고 깊은 니치의 포지셔닝

경쟁은 크게 세 축으로 구분된다. ① aquaculture computer vision 업체들 ② underwater ROV·stock monitoring 장비 기업들 ③ 대형 수산기업의 인하우스 솔루션. OctaPulse는 이 중 hatchery QA/QC에 특화된 juvenile grading & deformity inspection에 집중하는 차별 포지셔닝을 취한다.

구분 OctaPulse 인접 경쟁군
주력 문제 치어·유어 품질검사, 기형 판별, phenotyping 수중 점검, cage inspection, biomass monitoring
기술 축 CV + modular hardware + robotics ROV, underwater cameras, general analytics
대표 업체 OctaPulse (YCW26) Aquabyte, Blueye Robotics, Qysea
도입 가치 검사시간 단축, 선별 정확도, 데이터 자산화 운영 모니터링, 구조물 점검, 생체량 추정
확장 경로 hatchery → feeding → health → processing 장비 활용 범위 넓지만 QA 특화 약함
반복 작업 빈도 높음 (매 코호트, 개체 단위) 중간 (장비 점검·모니터링 주기)

OctaPulse는 수중 점검보다 더 높은 빈도의 반복 작업에 위치하며, ROI가 증명되면 빠른 확산이 가능한 포지션이다. 단, 대형 양식 기업이나 장비업체가 유사 기능을 내재화할 경우, 데이터 축적과 워크플로 통합 속도로 차별성을 유지해야 한다.

05 사업모델과 상업화 단계

현재 확인되는 매출 신호는 유료 파트너십과 6자리(USD) 연간 계약이다. PoC 단계를 넘어 상용 파일럿에서 도입 가치를 검증 중이라는 의미이며, 초기 B2B 딜 구조로는 긍정적이다.

💰 수익 구조 로드맵
Phase 1 (현재) — 하드웨어 설치 + 초기 계약금 / 유료 파일럿
Phase 2 — 구독형 소프트웨어 + 모델 업데이트 서비스 (Recurring)
Phase 3 — species dataset 확장 서비스 + 운영 플랫폼 (Multi-farm SaaS)
장기 — 양식장 운영 OS화 → feeding / health monitoring / breeding 통합 플랫폼

양식장은 한 번 구축하면 워크플로 통합 비용이 크기 때문에, 고객 유지와 업셀링 가능성이 비교적 높다. 하드웨어 마진만으로 평가하기보다 소프트웨어 구독 + 데이터 기반 고도화로 확장하는 전략이 합리적이다.

06 투자 포인트 3가지
✅ Bull Case #1 — 명확한 ROI 언어
검사시간 5분 → 30초 미만, 정확도 95%+. 고객이 즉시 계산 가능한 ROI 언어로 세일즈가 가능하다. labor cost 절감 + throughput 개선은 양식장 P&L에 직접 반영된다.
✅ Bull Case #2 — 북미 최대 trout 업체 레퍼런스
Riverence와의 6자리 연간 계약은 PoC를 넘어 상업적 검증이다. 보수적인 양식업계에서 tier-1 레퍼런스는 후속 고객 확보의 핵심 신호이며, 파이프라인 가속도를 만든다.
✅ Bull Case #3 — 데이터 기반 진입장벽 구축 가능성
실제 현장 데이터가 누적될수록 species별 일반화와 성능 개선이 가능해진다. 단순 장비 업체 → 운영 플랫폼으로의 진화 경로가 명확하며, deployment 확대 = 모이트 확대 구조가 성립한다.
07 주요 리스크 매트릭스
HIGH
현장 배포 복잡성
조도·수분·물 상태·개체 움직임 변수 → 데모 환경과 실운영 성능 격차 가능성. 모델 정확도보다 반복 배포 안정성이 핵심.
HIGH
고객 도입 사이클
양식업은 보수적 산업. 기술 우수성만으로 빠른 대규모 확산이 보장되지 않는다. 의사결정 사이클 길고 검증 기간 소요.
MID
Species 일반화 성능
trout에서 검증된 모델이 salmon/red drum/yellowtail 등 다종에서 동일 성능을 내는지는 추가 검증 필요.
MID
대형 업체 내재화 위험
Mowi, Cermaq 같은 대형 수산기업이나 장비업체가 유사 기능을 인하우스 개발할 경우 경쟁 심화. 데이터 축적 속도가 방어선.
MID
Gross Margin 구조
하드웨어 의존 초기 단계에서는 margin이 낮을 수 있다. SaaS 전환 속도와 하드웨어 원가 절감이 중장기 수익성의 핵심 변수.
LOW
시장 실재성
$300B 글로벌 양식업 시장은 실재하며, 검사 자동화 수요는 명확하다. 시장 존재 자체에 대한 리스크는 낮음.
08 투자 의견 및 Due Diligence 체크리스트
■ 종합 의견
OctaPulse는 초기 단계임에도 문제 정의 명확성, 상용 고객 검증, 데이터 기반 확장성 측면에서 유망한 초기 B2B AI/Robotics 투자 후보로 평가된다. 특히 오프라인 운영 최적화 여지가 큰 양식업에서 현장 통합형 솔루션은 단순 소프트웨어보다 더 큰 가치 창출이 가능하다.

현 단계에서 핵심 판단 기준은 "보여준 성능"이 아니라 "지속적으로 반복 재현 가능한 성능"이다.
