① The Great Parallel — 로보틱스는 LLM의 4단계 진화 경로(사전학습→정렬→추론→자율연구)를 똑같이 걷고 있다. 소프트웨어 AI가 14년 걸린 여정을 로보틱스가 훨씬 빠르게 반복 중이다.
② VLA의 시대는 끝났다 — 언어 모델에 액션 헤드를 얹은 VLA 대신, 물리 법칙을 직접 꿈꾸는 World Action Model(WAM)이 새 패러다임. NVIDIA DreamZero가 선두.
③ 데이터 병목 해결 — 텔레오퍼레이션(인간 연료)은 0.1%로 축소. EgoScale의 인간 1인칭 영상 + DreamDojo 신경망 시뮬레이터가 병목을 돌파. "컴퓨트 = 환경 = 데이터"

"언어 우선" 아키텍처

  • 언어 모델에 액션 헤드를 얹은 구조 ("Head-heavy")
  • "명사(Nouns)" 인식엔 강함 — 테일러 스위프트 옆 콜라캔 ✓
  • "동사(Verbs, 물리 법칙)"에 취약 — 중력, 마찰, 충격 ✗
  • 언어 공간에서 물리 세계를 이해하려는 근본적 불일치
  • 대표 모델: GR00T N1.5, π0.7
✅ 신식: WAM (World Action Model)

"비디오 우선" 아키텍처

  • 다음 토큰이 아닌 다음 물리 상태를 예측
  • 미래 픽셀 + 관절 토크를 동시에 "꿈꾸는(Dream)" 방식
  • Sora, Veo에서 물리 법칙이 자연 창발됨이 증명
  • 시각·행동이 1등급 시민 (언어는 보조)
  • 대표 모델: NVIDIA DreamZero (14B 파라미터)
데이터 전략 혁명 — 텔레오퍼레이션의 죽음
🪦 구식: 텔레오퍼레이션 (죽음)
  • 하루 최대 24시간 물리적 한계
  • "인간 연료(Human Fuel)" — 화석연료보다 희귀
  • 로봇 하드웨어 오작동으로 수율 저하
  • 수백만 시간 데이터 = 수천 년 필요
  • 현재 학습 비중: <0.1%
🚀 신식: 센서화된 인간 데이터
  • UMI — 손에 착용, 로봇 없이 데이터 수집
  • EgoScale — 1인칭 영상 20,854시간
  • 로그-선형 스케일링 법칙 발견
  • 데이터 증가 → 덱스테러티 단조 증가
  • 내년 내 1,000만 시간 목표
