# 📥 _inbox — 처리 대기

AI 처리 전 원문을 임시 보관하는 공간.
정리하지 말고 일단 던져두세요. Claude가 처리합니다.

## 사용법

1. 읽은 기사 URL, PDF, 회의 메모, 스크린샷을 여기에 모은다
2. 매주 월요일, Claude에게 요청:
   > "inbox 폴더에 있는 내용을 처리해서 각 카테고리에 배치해줘"
3. 처리 완료된 파일은 삭제 또는 `_done/` 으로 이동

## 파일 명명 규칙

```
YYYY-MM-DD_제목_간략히.md
예) 2026-05-19_OpenAI_o3_출시소식.md
```
