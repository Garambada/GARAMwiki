# 📔 06_나의_인사이트

AI가 못 채우는 부분. 오직 나만의 생각과 경험.
**업데이트 주기: 수시로 (즉흥적 아이디어 포착)**

## 핵심 파일

- `_의사결정_로그.md` — AI 관련 결정을 날짜순으로 기록
- `_실험_기록.md` — 직접 써본 것, 실패한 것
- `_아이디어_플로우.md` — 아직 검증 안 된 아이디어 모음

## 사용 원칙

1. **완성도 무시**: 생각이 떠오르면 즉시 기록. 나중에 다듬기
2. **AI 요약 금지**: 이 폴더만큼은 내 말로 직접 쓰기
3. **결정의 맥락 보존**: "무엇을 결정했나"보다 "왜 그렇게 결정했나" 기록

## 1분 일일 루틴

```
오늘 들은 LLM 키워드 → 용어사전에 추가
오늘 내린 AI 관련 결정 → 의사결정 로그 한 줄
오늘 흥미로운 발견 → 해당 카테고리 파일에 링크
```
