Overview

Hermes Agent란 무엇인가

Nous Research가 2026년 2월 25일 공개한 Hermes Agent는 단순한 챗봇이나 코드 실행 에이전트가 아니다. 사용자와 장기적 관계를 맺으며 대화 세션을 넘어 지속적으로 성장하는 자율 AI 에이전트다. 핵심 차별점은 "Closed Learning Loop" — 에이전트가 도구를 사용할수록 자동으로 SKILL.md를 생성하고, 사용자 선호를 USER.md에 업데이트하고, 중요 사실을 MEMORY.md에 기록한다.

기존 에이전트 프레임워크(LangChain, AutoGPT 등)가 개발자 중심의 복잡한 파이프라인 구성을 요구했다면, Hermes Agent는 메시지 앱에서 자연어 대화만으로 작동하며 20개 이상의 플랫폼(텔레그램, 디스코드, 슬랙, WhatsApp 등)을 공식 지원한다.

"Hermes Agent isn't just an AI assistant — it's a living system that evolves with every interaction, building a personalized model of you and developing skills tailored to your workflow."

— Nous Research, GitHub README, 2026

Architecture

핵심 아키텍처: 파일 기반 메모리 시스템

Hermes Agent의 "성장" 메커니즘은 4개의 핵심 마크다운 파일로 구현된다. 벡터DB나 복잡한 임베딩 없이, 플레인 텍스트 파일 + FTS5 전문 검색으로 장기 기억을 실현한다.

SOUL.md
에이전트의 성격·어조·가치관 정의. 시스템 프롬프트 역할. 사용자가 직접 커스터마이징 가능. 에이전트의 "정체성 헌장".
MEMORY.md
사용자에 대한 지속 기억. 선호도, 주요 결정사항, 반복 패턴, 중요 사실. FTS5 검색으로 세션 간 연속성 유지.
USER.md
사용자 프로파일. Honcho dialectic 모델링으로 대화에서 사용자 의도·선호·작업 스타일을 추론·업데이트.
SKILL.md
도구를 5회 이상 사용 후 자동 생성되는 절차적 스킬. agentskills.io 호환, 포터블·공유 가능. "에이전트가 쓴 자기 매뉴얼".
.hermes.md
글로벌 컨텍스트 파일. 프로젝트 전반 인식 정보 보관. 에이전트가 현재 환경을 이해하는 데 활용.
AGENTS.md
Claude Code 방식의 멀티-에이전트 컨텍스트. 서브에이전트 위임·조율을 위한 설정 파일.

Model Stack

모델 지원 & Hermes 4.3

Hermes Agent는 완전한 모델 무관(Model-Agnostic) 설계. 어떤 LLM도 백엔드로 사용 가능하다. 동시에 Nous Research는 자체 에이전트 최적화 모델 Hermes 4.3을 개발했다.

모델/제공자특징적합 용도
Hermes 4.3 ByteDance Seed 36B 기반. Psyche 분산 학습 네트워크로 훈련. 에이전트 태스크 최적화. Hermes Agent 네이티브 최적 성능
Claude (Anthropic) 강력한 추론·코딩·한국어 성능 복잡한 분석, 문서 작성
OpenAI GPT-4o 멀티모달, 범용 이미지 분석, 다양한 태스크
Google Gemini 멀티모달, 긴 컨텍스트 대용량 문서 처리
Groq (LPU) 초고속 추론 실시간 대화형 태스크
Ollama (로컬) 완전 오프라인, 데이터 보안 기밀 데이터, 프라이버시 중시
OpenRouter (200+) 모든 오픈소스 모델 접근 비용 최적화, 실험

Psyche 분산 학습 네트워크

Hermes 4.3은 Nous Research의 Psyche 분산 훈련 네트워크로 학습된 첫 번째 모델이다. 여러 기관의 GPU를 분산 활용하여 대규모 모델을 탈중앙화된 방식으로 훈련하는 실험적 인프라. ByteDance의 Seed 36B 모델을 기반으로 에이전트 태스크에 특화 파인튜닝했다.

Installation

설치 및 시작

MIT 라이선스 오픈소스. 단일 curl 명령으로 설치 완료.

빠른 설치 (macOS / Linux)
curl -fsSL https://raw.githubusercontent.com/NousResearch/hermes-agent/main/scripts/install.sh | bash
시작 명령
hermes --platform telegram --model claude-opus-4-6
📋 요구사항
Python 3.10+, Node.js 18+, LLM API 키 (또는 Ollama 로컬). 별도 서버 불필요.
⚙️ 설정 파일
~/.hermes/config.yaml에서 플랫폼·모델·메모리 경로 설정. SOUL.md는 성격 커스터마이징.
🔗 agentskills.io
커뮤니티 SKILL.md 마켓플레이스. 다른 사용자의 스킬을 임포트하거나 자신의 스킬을 공유.

C-Level 관점: Hermes Agent가 Founders Studio에 시사하는 것

🎯
Company Builder의 관점에서 본 Hermes Agent
1
개인 AI 에이전트 = 새로운 CXO 역량
Hermes Agent가 보여주는 방향은 명확하다: AI는 도구가 아니라 함께 성장하는 파트너다. SOUL.md에 회사 전략·가치관을 정의하고, MEMORY.md에 포트폴리오 컨텍스트를 축적하면, 에이전트는 시간이 갈수록 더 뛰어난 전략 파트너가 된다. Founders Studio의 C-레벨 업무에 즉시 적용 가능하다.
2
SKILL.md 자동화 = 비즈니스 프로세스 자동화의 새로운 패러다임
반복 업무 5회 실행 → SKILL.md 자동 생성 → 이후 한 문장 명령으로 실행. SOP(표준운영절차)가 에이전트 스킬로 자동 변환되는 개념이다. Founders Studio의 반복 투자 심사, 포트폴리오 리포팅, 채용 프로세스에 적용하면 운영 효율이 획기적으로 향상된다.
3
오픈소스 + MIT = 스타트업 친화적 에이전트 인프라
클로즈드 SaaS 에이전트 구독료 없이, 자체 서버에 배포하고 데이터를 완전 통제할 수 있다. 투자 정보, 포트폴리오 데이터, 전략 문서를 외부 클라우드에 노출하지 않고 Ollama 로컬 LLM과 결합하면 기밀 유지와 AI 활용을 동시에 달성한다.
4
텔레그램/슬랙 통합 = 팀 AI 에이전트 즉시 배포
이미 팀이 사용 중인 슬랙·텔레그램에 Hermes Agent를 연결하면, 팀 전체가 공유하는 AI 에이전트를 별도 앱 설치 없이 운영한다. 스타트업 포트폴리오사에게 이 에이전트를 온보딩하면 Founders Studio만의 차별화된 지원 체계가 된다.
5
Hermes 4.3 × Psyche 네트워크 = 분산 AI 훈련의 상업화
ByteDance Seed 36B를 Psyche 분산 네트워크로 학습시킨 Hermes 4.3은 탈중앙화 AI 훈련의 실제 구현 사례다. 미래의 스타트업 스튜디오 모델에서 "공유 AI 인프라"를 사업화할 때 참고할 선례다. 분산 컴퓨팅 × AI 훈련 = 새로운 투자 카테고리.
Use Cases

실제 활용 시나리오

📊
포트폴리오 모니터링 에이전트
텔레그램으로 "이번 주 포트폴리오 업데이트 요약해줘" → 웹 검색 + MEMORY.md 컨텍스트 → 자동 리포트 생성. 시간이 갈수록 각 포트폴리오사의 특성을 학습해 리포트 품질 향상.
🔍
투자 리서치 자동화
딜소싱 후 "이 스타트업 분석해줘" → 웹 크롤링 + 코드 실행 + 문서 분석 → SKILL.md에 투자 심사 루틴 저장 → 다음 딜부터 원클릭 실행.
📝
전략 문서 코파일럿
SOUL.md에 회사 전략 기술 → 슬랙에서 "사업계획서 초안 잡아줘" → 이전 대화에서 학습한 선호 스타일로 맞춤 작성. 반복할수록 더 정확.
🤝
포트폴리오사 지원 봇
각 포트폴리오 스타트업에 Hermes Agent 배포 → 슬랙 연동 → 창업자들의 반복 질문(법무·회계·HR) 자동 대응. Founders Studio의 킬러 부가가치 서비스.
🔒
프라이빗 AI 워크스테이션
Ollama 로컬 LLM + Hermes Agent → 모든 데이터 온프레미스. 투자 정보·M&A 정보 처리 시 완전 기밀 유지. 외부 API 없는 폐쇄망 운영.
📱
모바일 퍼스트 AI 에이전트
이동 중 WhatsApp/텔레그램에서 AI 에이전트 활용. 스마트폰만으로 코드 실행, 파일 처리, 웹 리서치. CEO 이동성에 최적화.
Conclusion

결론: 에이전트 AI의 "성장" 패러다임 전환

Hermes Agent가 증명하는 것: AI 에이전트의 미래는 '더 강력한 기능'이 아니라 '함께 성장하는 관계'다. SKILL.md가 자동 생성되고 MEMORY.md가 쌓이고 USER.md가 진화할수록, 에이전트는 단순 도구에서 조직 지식을 체화한 디지털 파트너로 변모한다.

— GARAMbada 분석, 2026.05
